Iteration using for loop

for loop can be used to repeatedly execute a block of code.

1for i in range(5):
2    print("Hello")
Output
Hello
Hello
Hello
Hello
Hello

1for i in range(5):
2    print(i)
Output
0
1
2
3
4

What happened in the code above?

  • range() function can produce a sequence of integers 0,1,2,3,40, 1, 2, 3, 40,1,2,3,4.
  • for loop allows us to iterate i.e. “go over” that sequence, a number at a time
    • In each step of the loop, variable i gets a value from the sequence
  • We can have any valid variable name, other than i if we want.

Using range() function

range() function takes up to 3 arguments:

  • range(end) — produces sequence 0, 1, 2, ..., end-1
  • range(start, end) — produces sequence start, start+1, ..., N where N < end
  • range(start, end, step) — if step > 0, produces sequence start, start+step, ..., N where N < end — if step < 0, produces sequence start, start+step, ..., N where N > end
1# 0, 1, 2, ..., 9
2for i in range(10):
3    print(i)
4

5# 1, 2, ..., 10
6for i in range(1, 11):
7    print(i)
8    
9# 0, 2, 4, ..., 18    
10for i in range(0, 20, 2):
11    print(i)
12

13# 10, 15, 20, 25, ..., 95
14for i in range(10, 100, 5):
15    print(i)
16

17# 10, 9, 8, ..., 1
18for i in range(10, 0, -1):
19    print(i)

Examples

  1. Compute sum of first N numbers.
1N = 50
2

3total = 0
4for num in range(1, N+1):
5    total = total + num
6    
7print(total)
  1. Compute sum of all multiples of either 3 or 5, between 100 and 200 (both inclusive).
1total = 0
2for n in range(100, 201):
3    if n % 3 == 0 or n % 5 == 0:
4        total = total + n
5    
6print(total)

Indexing & Slicing Strings

Recall that a string is a sequence of characters.

Each character, therefore, has a position or an index.

Index starts with zero. For example, for the string "Hello":

Hello
01234

Indices must be integers and cannot be float.

Python also allows negative indices, which go from right to left:

Hello
-5-4-3-2-1

For any string s,

  • valid positive index values are from 0 to len(s)-1.
  • valid negative index values are from -len(s) to -1.

Square brackets [] are used to get letter in a string at given index.

1>>> message = "Hello"
2>>> message[0]  # first letter
3'H'
4>>> message[1]  # second letter
5'e'
6>>> message[4]  # fifth letter, the last one in the string
7'o'
8>>> message[5]  # there is no letter at this index
9IndexError: string index out of range
10>>> message[-1]
11'o'
12>>> message[-5]
13'H'
14>>> message[-6]  # there is no letter at this index
15IndexError: string index out of range
16>>> message[1.0]
17TypeError: string indices must be integers

Traversing a string

We can use for loop with range() function to go over a string letter-by-letter.

1message = "Hello"
2

3for i in range(len(message)):
4    print(i, message[i])
Output
0 H
1 e
2 l
3 l
4 o

A more interesting example:

1letters = "bcmrst"
2

3for i in range(len(letters)):
4    print(letters[i] + "ake")

Using slice to get substrings

Using slice notation we can get parts of a string.
string[start:end:step]

Indices must be integers and begin at zero. start, end, step values work similar to range() function.

1>>> fruit = "pineapple"
2>>> fruit[4:7]  # letters at indices 4, 5, 6 (7 is not included)
3'app'
4>>> fruit[:4]  # start index is 0 if omitted, same as fruit[0:4]
5'pine'
6>>> fruit[4:]  # end index is len(fruit) if omitted, same as fruit[4:len(fruit)]
7'apple'
8>>> fruit[2:7:2]  # select letters at indices 2, 4, 6
9'nap'
10

11>>> fruit[-5:]  # start and index -5 and go to the end of string
12'apple'
13>>> fruit[-5:-2]  # select letters at -5, -4, -3  (-2 is not included)
14'app'
15

16# Here the third number -1 is the step size
17# Negative step size mean go from right to left, i.e. in reverse order
18>>> fruit[-1:-6:-1]  # Select letter at -1, -2, -3, -4, -5  (-6 is not included)
19'elppa'
20

21# Omitting start and end mean select whole string
22# But step size -1 means go from right to left i.e. in reverse order 
23>>> fruit[::-1]
24'elppaenip'
