List methods

Python has several methods that we can call on a list object:

1# list.append(x): 
2#   Adds the item x to the end of the list.
3grades = [90, 70, 60.5, 70, 80]
4grades.append(100)
5print(grades)  # [90, 70, 60.5, 70, 80, 100]
6

7grades.append("30")
8print(grades)  # [90, 70, 60.5, 70, 80, 100, '30']
9

10grades.append(False)
11print(grades)  # [90, 70, 60.5, 70, 80, 100, '30', False]
12

13

14# list.insert(i, x): 
15#   Adds the item x to the list at index i
16grades = [90, 70, 60.5, 70, 80]
17grades.insert(0, 100)
18print(grades)  # [100, 90, 70, 60.5, 70, 80]
19

20# insert() works even when index is greater than length of list
21grades = [90, 70, 60.5, 70, 80]
22grades.insert(10, "B+")
23print(grades)  # [90, 70, 60.5, 70, 80, 'B+']
1# list.remove(x):

5grades.remove(70)
6print(grades)  # [90, 60.5, 70, 80, 'B+']

8grades.remove("B+")

15# list.pop(i):

Lists and Strings

There are built-in functions and string methods that allows us to transform strings to/from lists.

1# list(seq): built-in function which converts a sequence (such as
2#   a string or a list) into a list
3fruit = "apple"
4letters = list(fruit)
5print(letters)  # ['a', 'p', 'p', 'l', 'e']
6

7

8# s.split(sep): Breaks the string s using the separator string sep
9#   and returns a list of strings containing the separated parts
10# s.split(): same as above but using whitespace 
11#   (spaces, tab character and newline ) as separator
12data = "Khalid Reza Deven"
13names = data.split()
14print(names)  # ['Khalid', 'Reza', 'Deven']
15

16# Multiple spaces are also removed
17data = "Khalid    Reza     Deven"
18names = data.split()
19print(names)  # ['Khalid', 'Reza', 'Deven']
20

21data = "Khalid\tReza\tDeven" # separated by tab
22names = data.split()
23print(names)  # ['Khalid', 'Reza', 'Deven']
24

25data = """Khalid
26Reza
27Deven Parekh"""
28names = data.split()
29print(names)  # ['Khalid', 'Reza', 'Deven', 'Parekh']
1# s.splitlines(): Breaks a multi-lines strings into separate lines
2#   and returns a list containing those lines.

4Reza
