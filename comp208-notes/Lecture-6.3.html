6.3 — Dictionaries

Dictionaries

Motivation

Suppose we would like to store the following enrollment data:

semesterno. of students
F2017816
W2018613
F2018709
W2019590
F2019744

We can do this using two lists for the two columns:

1semesters = ['F2017', 'W2018', 'F2018', 'W2019', 'F2019']
2students = [816, 613, 709, 590, 744]

What should we do if we want to add new data?

1semesters.append("F2020")
2students.append(550)
3# ['F2017', 'W2018', 'F2018', 'W2019', 'F2019', 'F2020']
4# [816, 613, 709, 590, 744, 550]

What if we want to modify the value for a specific semester?

1idx = semesters.index("W2018")
2students[idx] = 600

What we if try to add an entry for a semester that already exists?
List allows duplicates so it does not check if a semester already exists.

Use a dictionary!

  • You can think of an item of a dictionary as a pair of objects:
    • The first object of the pair is called a key.
    • The second object is referred to as the value.
  • A dictionary is called a mapping type because it maps key objects to value objects.
1# A dictionary is created using a sequence of key-value pairs
2enrollment = {'F2017': 816, 'W2018': 613, 'F2018': 709,
3              'W2019': 590, 'F2019': 744}
4print(type(enrollment))  # <class 'dict'>
5

6# Number of key-value pairs
7print(len(enrollment))  # 5
8

9# This is an empty dictionary, not a set!
10empty_dict = {}
11print(len(empty_dict)) # 0

Note on keys and values

  • Keys
    • Have to be immutable objects.
    • Have to be unique in a dictionary. A dictionary cannot contain two items with the same key.
  • Values
    • Values can be of any type; both mutable and immutable values are allowed.
    • Many keys can map to the same value. i.e. values need not be unique.

Dictionary Lookup

With lists, we can access an item of the list through its index.

With dictionaries, we can access a value stored in the dictionary through the key associated with it.

1enrollment = {'F2017': 816, 'W2018': 613, 'F2018': 709,
2              'W2019': 590, 'F2019': 744}
3

4

5num_students = enrollment["F2018"]
6print(num_students)  # 709
7

8# Key must exist in the dictionary if we want to access its value
9print(enrollment["F2020"])  # KeyError: 'F2020'

Adding an item

We can add a new item by specifying a key and a value: dictionary[key] = value

1# A dictionary is created using a sequence of key-value pairs
2enrollment = {'F2018': 709, 'W2019': 590, 'F2019': 744}
3

4

5# add an item
6enrollment["F2020"] = 800
7

8# add another item
9enrollment["W2020"] = 900
10

11print(enrollment)
12# {'F2018': 709, 'W2019': 590, 'F2019': 744, 'F2020': 800, 'W2020': 900}
13

14# change an existing item
15enrollment["F2018"] = 700
16

17print(enrollment)
18# {'F2018': 700, 'W2019': 590, 'F2019': 744, 'F2020': 800, 'W2020': 900}

Removing an item

We can delete an item using the following syntax:
del dictionary[key]

1enrollment = {'F2018': 709, 'W2019': 590, 'F2019': 744}
2

3

4del enrollment["F2019"]
5

6print(enrollment)
7# {'F2018': 709, 'W2019': 590}
8

9del enrollment["F2020"]
10# KeyError: 'F2020'

Check for membership

We can check if a key is part of a dictionary using the in and not in operators.

1d = {'x' : 0, 'y' : 1, 'z' : 2} 
2print('x' in d) #  True 
3print(0 in d) #  False
4print(0 not in d) #  True

Iterating through a dictionary

We can use a for loop to iterate through all the keys in a dictionary.

1enrollment = {'F2018': 709, 'W2019': 590, 'F2019': 744}
2

3for key in enrollment:
4    print(key, "->", enrollment[key])
Output
F2018 -> 709
W2019 -> 590
F2019 -> 744

Functions and methods for dictionaries

1# dict(L): creates and returns a dictionary using a list L of tuples, 
2#   where each tuple is of length 2 in form of (key, value).
3pairs = [("Montreal", 1.78), ("Rome", 2.87), ("Tokyo", 9.27)]
4d = dict(pairs)
5print(d)
6#  {'Montreal': 1.78, 'Rome': 2.87, 'Tokyo': 9.27}
7print(d["Rome"])
8# 2.87
9

10

11# dict.keys(): returns a iterable (sequence) of all keys
12keys = list(d.keys())
13print(keys)  # ['Montreal', 'Rome', 'Tokyo']
14

15

16# dict.values(): returns a iterable (sequence) of all values
17values = list(d.values())
18print(values)  # [1.78, 2.87, 9.27]
19

20

21# dict.items(): returns a iterable (sequence) of tuples (key, value) 
22#   for all items in the dictionary
23d = {'Montreal': 1.78, 'Rome': 2.87, 'Tokyo': 9.27}
24pairs = list(d.items())
25print(pairs)
26# [('Montreal', 1.78), ('Rome', 2.87), ('Tokyo', 9.27)]

For more methods, help help(dict).

Using the above methods to iterate over a dictionary:

1d = {'Montreal': 1.78, 'Rome': 2.87, 'Tokyo': 9.27}
2total = 0
3for val in d.values():
4    total += val
5

6print(total)  # 13.92

Making use of tuple unpacking in the for loop:

1d = {'Montreal': 1.78, 'Rome': 2.87, 'Tokyo': 9.27}
2

3# d.items() returns a sequence of key-value tuples
4for tup in d.items():
5    key = tup[0]
6    val = tup[1]
7    print(key, "->", val)
8

9# Montreal -> 1.78
10# Rome -> 2.87
11# Tokyo -> 9.27
12

13

14# Unpack tuple into two variables: key and val
15for key, val in d.items():
16    print(key, "->", val)
17

18# Montreal -> 1.78
19# Rome -> 2.87
20# Tokyo -> 9.27
