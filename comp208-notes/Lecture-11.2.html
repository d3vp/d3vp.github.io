    • passes through all the points exactly (interpolation)

How interpolation and curve-fitting are useful?

  • If we can find a function f(x)f(x)f(x) that interpolates or approximates the data well,
    • We can try to predict new data points (xk,yk)(x_k, y_k)(xk​,yk​).
    • We can better understand the underlying process that might have generated the data.

Polynomial interpolation

  • The simplest form of interpolation function is a polynomial function.
  • It is always possible to construct a polynomial P(x)P(x)P(x) of degree nnn that passes through n+1n+1n+1 data points.

Curve fitting

  • If data is obtained experimentally, there could be random noise present due to measurement errors.
    • Interpolation would not be a good choice as it would fit all data points including noise as well.
  • Instead, we use curve fitting, to find a smooth curve that fits the “average” of the data points, and that is less affected by the noise or “outlier” points.

  • Linear regression is a simpler case of curve fitting where a straight line is found that fits a set of data points.
  • Unlike interpolation, the line need not pass through the points.
  • Linear regression line is given by f(x)=ax+bf(x) = ax + bf(x)=ax+b, where aaa is the slope of the line, and bbb is the y-intercept.
  • The main task will be to find aaa and bbb such that the line “fits” the data points in the “best” possible way.

How to determine the “best” fit?

  • To know how good a line fits the given data points, we compute error for each data point.

  • Error is defined as the squared difference between the actual y-value yiy_iyi​ and the y-value given by the regression line, f(xi)f(x_i)f(xi​).

The total error EEE for all data points is given by

E(a,b)=∑i=0n(yi−f(xi))2=∑i=0n(yi−(axi+b))2 \begin{aligned} E(a, b) &= \sum_{i=0}^{n} \left ( y_i - f\left ( x_i \right ) \right ) ^ 2 \\ &= \sum_{i=0}^{n} \left ( y_i - \left ( ax_i + b \right ) \right ) ^ 2 \end{aligned}E(a,b)​=i=0∑n​(yi​−f(xi​))2=i=0∑n​(yi​−(axi​+b))2​

One way to define “best” fit:

  • Given all the data points (xi,yi)(x_i, y_i)(xi​,yi​), we want to find a line (i.e. find aaa and bbb) such that the error E(a,b)E(a, b)E(a,b) is minimized.

To find aaa and bbb that minimize E(a,b)E(a, b)E(a,b), we calculate the partial derivatives of EEE with respect to aaa and bbb and solve for a and b when derivates are zero:

∂E∂a=0,     ∂E∂b=0 \frac{\partial E}{\partial a} = 0,\, \, \, \, \, \frac{\partial E}{\partial b} = 0∂a∂E​=0,∂b∂E​=0

Calculating the partial derivatives

∂E∂a=∑i=0n2(yi−(axi+b))(−xi)\frac{\partial E}{\partial a} = \sum_{i=0}^{n} 2 (y_i - (ax_i + b)) (-x_i)∂a∂E​=i=0∑n​2(yi​−(axi​+b))(−xi​)
∂E∂b=∑i=0n2(yi−(axi+b))\frac{\partial E}{\partial b} = \sum_{i=0}^{n} 2 (y_i - (ax_i + b))∂b∂E​=i=0∑n​2(yi​−(axi​+b))

a=∑(xi−xˉ)(yi−yˉ)∑(xi−xˉ)2a = \frac{\sum (x_i - \bar{x}) (y_i - \bar{y}) } {\sum (x_i - \bar{x})^2 }a=∑(xi​−xˉ)2∑(xi​−xˉ)(yi​−yˉ​)​

where xˉ\bar{x}xˉ and yˉ\bar{y}yˉ​ are the averages of x-values and y-values of data points.

Finally, using these aaa and bbb, we get the desired best-fit line as f(x)=ax+bf(x)=ax+bf(x)=ax+b. Here aaa is slope of the line and bbb is the y-intercept.

Definite Integral

  • The definite integral of a function of a single variable, f(x)f(x)f(x), between two limits aaa and bbb can be viewed as the area SSS under the curve.

  • Numerical integration algorithms try to estimate this area

Numerical Integration

Our approach will be as follows:

  • Divide the region between aaa and bbb into NNN segments
  • We then estimate the area under the curve in each segment
  • Finally, we sum these areas

We will consider three algorithms for estimating this area in each segment:

  • The Midpoint method
  • The Trapezoidal method
  • Simpson’s method

Example

We will use the following function in the interval [0,5][0, 5][0,5] to compare the three algorithms:

The Midpoint Method

  • We first divide the region from aaa to bbb into NNN equal rectangular segments

  • The width of each segment is Δx=b−aN\Delta x = \frac{b-a}{N}Δx=Nb−a​

  • The endpoint of the segments are

    x0=a<x1<⋯<xN−1<xN=b \small x_0 = a < x_1 < \cdots < x_{N-1} < x_{N} = bx0​=a<x1​<⋯<xN−1​<xN​=b

    where

    xi=a+i⋅Δx,i∈[0,N] \small x_i = a + i \cdot \Delta x, \quad i \in [0, N]xi​=a+i⋅Δx,i∈[0,N]

  • We estimate the area under the curve in each rectangular segment using the value of f(x)f(x)f(x) at the midpoint of each segment.

    areak=Δx⋅f(xk+Δx2) , k∈[0,N) \small area_k = \Delta x \cdot f(x_k + \frac{\Delta x}{2})\ ,\ k \in [0, N)areak​=Δx⋅f(xk​+2Δx​) , k∈[0,N)

  • To compute an approximation to the integral, we just have to sum these areas

Approximation improves as the number of segments is increased.

Trapezoidal Method

To approximate the area of each segment, use the area of trapezoid rather than the rectangle.

The area of each trapezoidal segment is given by

areak=Δx2(f(xk)+f(xk+Δx)) , k∈[0,N) \small area_k = \frac{\Delta x}{2} \left( f(x_k) + f(x_k + \Delta x) \right)\ ,\ k \in [0, N)areak​=2Δx​(f(xk​)+f(xk​+Δx)) , k∈[0,N)

Simpson’s method

Given any three points there is a unique polynomial (parabola), called the interpolating polynomial, that passes through these points

  • Simpson’s method fits a parabola P(x)P(x)P(x) through the curve at three points — the value of the function at the two endpoints, and at the midpoint of the interval:

    f(xk), f(xk+Δx2), f(xk+Δx) \small f(x_k),\ f(x_k + \frac{\Delta x}{2}),\ f(x_k + \Delta x)f(xk​), f(xk​+2Δx​), f(xk​+Δx)

  • The area under the parabola in the interval [xk,xk+Δx][x_k, x_k + \Delta x][xk​,xk​+Δx] is given by:

    Δx6[f(xk)+4f(xk+Δx2)+f(xk+Δx)] \small \frac{\Delta x}{6} \left[f(x_k) + 4f(x_k + \frac{\Delta x}{2}) + f(x_k + \Delta x)\right]6Δx​[f(xk​)+4f(xk​+2Δx​)+f(xk​+Δx)]

  • Adding areas of all the NNN segments gives an approximation to the integral.

  • Simpson’s method generally finds a better approximation to the area under the curve in each segment than trapezoidal method which uses a line instead of parabola.

  • For more details on where does the formula above come from: https://en.wikipedia.org/wiki/Simpson%27s_rule

Accuracy of the three methods

  • For all the three methods we saw, as the number of subintervals NNN approaches infinity, approximation improves and approaches actual integral.

  • However, the speed of improvement differs:

    • Midpoint method — error∝1N2error \propto \frac{1}{N^2}error∝N21​
    • Trapezoidal method — error∝1N2error \propto \frac{1}{N^2}error∝N21​
    • Simpson’s method — error∝1N4error \propto \frac{1}{N^4}error∝N41​

  • Simpson’s method approaches the integral faster than other methods.
