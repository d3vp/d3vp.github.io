Packing vs Unpacking

When we create a string, a list, or a tuple, we are packing several elements into a single object.

1s = "cat"
2my_list = [5, 'a']
3my_tuple = (0, 3, 7)

Unpacking allows us to assign values in a string/list/tuple to multiple variables.
We must know the exact length of the string/list/tuple.

1>>> s = "cat"
2>>> a, b, c = s
3# a, b and c are all strings
4>>> print(a)
5c
6>>> print(b)
7a
8>>> print(c)
9t
10

11>>> my_list = [5, 'a']
12>>> x, y = my_list
13>>> print(x)  # an int
145
15>>> print(y)  # a string
16a
17

18# Parentheses are optional in this context.
19>>> my_tuple = 0, 3, 7
20>>> x, y, z = my_tuple
21>>> print(x)
220
23>>> print(y)
243
25>>> print(z)
267
27

28# Variables must match number of elements
29>>> tup = 1, 2, 3 
30>>> x, y = tup
31ValueError: too many values to unpack (expected 2)

Multiple assignment using packing/unpacking on same line

1# We are creating a tuple on the right side and 
2# unpacking it into 3 variables.
3>>> city, population, area = 'Montreal', 1704694, 431.5
4>>> print(city)

Returning a tuple from a function and unpacking

1def min_max(mylist):

4    
5

Functions and methods for dictionaries

1# dict(L): creates and returns a dictionary using a list L of tuples, 
2#   where each tuple is of length 2 in form of (key, value).
3>>> pairs = [("Montreal", 1.78), ("Rome", 2.87), ("Tokyo", 9.27)]
4>>> d = dict(pairs)
5>>> print(d)
6{'Montreal': 1.78, 'Rome': 2.87, 'Tokyo': 9.27}
7>>> print(d["Rome"])
82.87
9

10# dict.keys(): returns a iterable (sequence) of all keys
11>>> keys = list(d.keys())
12>>> print(keys)
13['Montreal', 'Rome', 'Tokyo']
14

15

16# dict.values(): returns a iterable (sequence) of all values
17>>> values = list(d.values())
18>>> print(values)
19[1.78, 2.87, 9.27]
20

21# dict.items(): returns a iterable (sequence) of tuples (key, value) 
22#   for all items in the dictionary
23>>> d = {'Montreal': 1.78, 'Rome': 2.87, 'Tokyo': 9.27}
24>>> pairs = list(d.items())
25>>> print(pairs)
26[('Montreal', 1.78), ('Rome', 2.87), ('Tokyo', 9.27)]

For more methods, help help(dict).

Using the above methods to iterate over a dictionary:
