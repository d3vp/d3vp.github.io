    Now, we look at a method get available on a dictionary object.
    dict.get(key, default_value) returns the value in dictionary associated with the given key if key is present in the dictionary, otherwise returns the default_value.

    1d = {"A": 2}
    2

    3# Key "A" exists in d so its associated value in d is returned
    4print(d.get("A", 0))  
    5# 2
    6

    7# Key 123 does not exist in d, so default value 0 is returned
    8print(d.get(123, 0)) 
    9# 0

  2. Write a function invert that takes a dictionary dct as argument and returns an inverted dictionary inv, where keys in dct become values in inv and values in dct become keys in inv. Since keys are unique in any dictionary but values need not be unique, the inverted dictionary will need to use a list for values. For example,

    1letter_counts = {"a": 1, "b": 1, "apple": 5, "bus": 3, "cat": 3, "dog": 3}
    2print(invert(letter_counts))
    Output
    {1: ['a', 'b'], 5: ['apple'], 3: ['bus', 'cat', 'dog']}

    In the above example, since letter_counts contains duplicate values such as 1, when those values become keys in the inverted dictionary, we will need a list, e.g. 1: ['a', 'b']

Write a function matrix_sum that takes two matrices (list of lists) as arguments, call them A and B, and returns a matrix C (list of lists), such that C is the element-wise sum of A and B i.e. C[i][j] = A[i][j] + B[i][j] for all i, j. You can assume that A and B will be of same size (no. of rows and columns will be same.)

    • You will need to first create a matrix C (list of lists) such that it contains all zeros.
      • To do so, suppose nrows in no. of rows and ncols is no of columns in A (or B, they are same size).
      • Then to create a matrix C with zeros, you first create and empty list and then append to it nrows number of lists, each of which contain ncols number of zeros.
    • Then using nested loops you perform the operation C[i][j] = A[i][j] + B[i][j] for all i, j
    1matrix1 = [[1, 2, 3],
    2        [4, 5, 6]]
    3matrix2 = [[1, 2, 3],
    4        [1, 1, 1]]
    5result = matrix_sum(matrix1, matrix2)
    6print(result)
    Output
    [[2, 4, 6], [5, 6, 7]]
