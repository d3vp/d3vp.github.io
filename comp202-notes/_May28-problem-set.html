  1. Persons A and B were married but are getting divorced. They need to split the objects they jointly owned. They have decided to each rank these objects in their order of preference (first = most preferred), and then assign each object to whoever had assigned it the highest rank. Objects that were assigned the same rank by both people will go to charity. For example, consider the following rankings,

    1A_ranking = ["car", "rug", "table", "tv", "lamp", "sofa"]
    2B_ranking = ["rug", "sofa", "tv", "table", "lamp", "car"]

    The program should print:

    Output
    Person A keeps: car, table
Person B keeps: rug, tv, sofa
Goes to Charity: lamp

    Assume that the lists A_ranking and B_ranking will both be of same length and will contain same objects but in different order.

    Here is one way to solve this problem:

    • Create 3 empty lists to keep track of which objects will go to A, B, or charity.
    • For each object x in A_ranking

      • Find the ranking of the object x in B_ranking (use the index() method).

      • Compare rankings of x, and add (i.e. append) x to either A, B or charity’s list.

  6. Write a function that takes as argument a list array and counts how many times each element appears in array. The function should return a dictionary where each item i.e. each key-value pair is such that the key is a unique element from array and the value is how many times that key appears in array. For example,

    1>>> d = count_occurrences([10, 12, 10, 100, 250, 100, 100])
    2>>> print(d)
    3{10: 2, 12: 1, 100: 3, 250: 1}
    4>>> d = count_occurrences(["apple", "banana", "apple", "orange"])
    5>>> print(d)
    6{'apple': 2, 'banana': 1, 'orange': 1}

    Hint:

    • Start with empty dictionary
    • For each element x in the array,
      • if x is not present in the dictionary as a key, add an item to the dictionary with x as key and 111 as value. (Since we just saw x for the first time we start counting it from 111)
      • If x exists already in the dictionary as a key, increment its value by 111.
