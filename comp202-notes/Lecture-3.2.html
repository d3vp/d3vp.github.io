L3.2 — Lists

Lists

  • A list is like a container that holds a sequence of arbitrary objects.
  • Objects in a list are called elements.
  • Lists are ordered! The order in which the elements are stored in the list matters.
  • Each element is associated to an index (index 000: first element, index 111: second element, etc.)

Creating a list

1# A list is created using square brackets, 
2# with each element separated by a comma.
3>>> prime_numbers = [2, 3, 5, 7, 11, 13]
4>>> print(prime_numbers)
5[2, 3, 5, 7, 11, 13]
6

7>>> print(type(prime_numbers))
8<class 'list'>

A list can contain elements of any type. For example we can have a list of strings:

1>>> days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
2

3# Number of elements in the list
4>>> len(days)
57
6

7>>> empty_list = []
8>>> len(empty_list)
90

A list can contain any number of elements, from zero to as many as the computer’s memory allows.

A list can contains objects of different types.

1# list with mixed types
2>>> numbers = [1, 'two', 3.75]

Elements of a list don’t need to be unique.

1# list with duplicate values
2>>> numbers = [5, "five", 5]

What can we do with a list?

  • Add elements to the list e.g. adding new contact on your phone
  • Access elements from the list e.g. search a contact on your phone
  • Modify elements inside the list e.g. changing grades of a student
  • Deleting elements from the list e.g. remove someone from a friend list

Indexing a list

We can access an element inside a list using indexing (square brackets), just as we did for strings.

1>>> days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
2>>> first_day = days[0]
3>>> second_day = days[1]
4>>> last_day = days[6]
5>>> print(first_day, second_day, last_day)
6Mon Tue Sun
7

8# No element at index 7
9>>> print(days[7])  
10IndexError: list index out of range
11

12>>> numbers = [1, 'two', 3.75]
13>>> print(numbers[0] + numbers[2])
144.75
15>>> print(numbers[0] + numbers[1])
16TypeError: unsupported operand type(s) for +: 'int' and 'str'

Slicing a list

Similar to strings, we can also get a sub-list — parts of a list — using slice notation.

1>>> days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
2>>> print(days[2:5])
3['Wed', 'Thu', 'Fri']
4

5>>> print(days[:6:2])
6['Mon', 'Wed', 'Fri']
7

8# Makes a copy of the list
9>>> print(days[:])
10['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
11

12# Makes a reversed copy of the list
13>>> print(days[::-1])
14['Sun', 'Sat', 'Fri', 'Thu', 'Wed', 'Tue', 'Mon']

Concatenation (+) and Replication (*)

The operators + and * work for lists as they worked for a string:

1# lists a and b are joined to produce a third list c:
2>>> a = [1, 2]
3>>> b = [10, 11, 12]
4>>> c = a + b
5>>> print(c)
6[1, 2, 10, 11, 12]
7>>> print(a + b)  # without extra variable c
8[1, 2, 10, 11, 12]
9

10>>> a = [1, 2]
11# resulting list consists of repeated elements of list a:
12>>> c = a * 3 
13>>> print(c)
14[1, 2, 1, 2, 1, 2]
15>>> print(a * 3)
16[1, 2, 1, 2, 1, 2]

membership operators: in and not in

The in and not in operators test for membership.

We can use them to test if an object is an element of a list.

1>>> a = [1, 2]
2>>> b = [10, 11, 12]
3>>> 1 in a
4True
5>>> 11 in a
6False
7>>> 5 not in b
8True
9

10>>> x = 3.14
11>>> x in a or x in b
12False

These operators very useful in simplifying code:

1# Instead of long conditions like this:
2if x == 5 or x == 7 or x == 10:
3    # do something
1# Now we can do:
2if x in [5, 7, 10]:
3    # do something

Lists and Loops

For loops are very useful for looping through all the elements in an list. For instance we could use a for loop to compute the sum of all the elements in the list grades:

1grades = [85, 78.5, 98, 75, 100]
2

3total = 0
4for i in range(5):
5   total += grades[i]
6

7avg = total / 5
8print(avg) # prints 87.3

Above code can be generalized!

1grades = [85, 78.5, 98, 75, 100]
2

3total = 0
4N = len(grades)  # Number of elements in grades list
5for i in range(N):
6   total += grades[i]
7

8avg = total / N
9print(avg) # prints 87.3

Alternatively, we can avoid using an index if we don’t need to use an index in our loop:

1grades = [85, 78.5, 98, 75, 100]
2

3total = 0
4N = len(grades)  # Number of elements in grades list
5for g in grades:
6   total += g
7

8avg = total / N
9print(avg) # # prints 87.3

Modifying the content of a list

Unlike strings, we can modify the content of a list after it has been created!

We can change a single element using its index and the assignment operator.

1>>> days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
2>>> days[0] = "Sun"
3>>> print(days)
4['Sun', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
5

6>>> days[7] = "No such day"
7IndexError: list assignment index out of range

Updating contiguous elements

We can modify several contiguous elements using slice assignment. The syntax of the slice assignment is as follow:

list_object[start:stop:step] = iterable

For now, think of iterable to be a sequence such as a list or a string.

Examples:

1>>> days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
2>>> print(days[1:3])
3['Tue', 'Wed']
4

5>>> days[1:3] = [5, 8]
6>>> print(days)
7['Mon', 5, 8, 'Thu', 'Fri', 'Sat', 'Sun']
8

9>>> print(days[1::2])
10[5, 'Thu', 'Sat']
11>>> days[1::2] = ["", "", ""]
12>>> print(days)
13['Mon', '', 8, '', 'Fri', '', 'Sun']

Nested Lists

An element of a list can be another list!
Such lists are called nested lists.

1>>> nested_list = [[1], [1, 2, 3], [1, 2]]
2>>> type(nested_list)
3<class 'list'>
4

5>>> print(nested_list[0])
6[1]
7>>> print(nested_list[1])
8[1, 2, 3]
9>>> print(nested_list[2])
10[1, 2]
11

12>>> print(type(nested_list[0]), type(nested_list[1]), type(nested_list[2]))
13<class 'list'> <class 'list'> <class 'list'>

Nested lists are useful to store data which come in form of a table or spreadsheet.

1#                   Name,   A1,  A2, A3
2student_grades = [["Khalid", 90, 95, 100], # Student 1
3                  ["Reza", 85, 90, 98], # Student 2
4                  ["Deven", 70, 75, 80]] # Student 3

We can perform same operations on nested lists as we saw earlier: indexing, slicing, etc.

1>>> student_grades = [["Khalid", 90, 95, 100],
2                      ["Reza", 85, 90, 98],
3                      ["Deven", 70, 75, 80]]
4

5# Print name of 2nd student
6>>> print(student_grades[1][0]) 
7Reza
8

9# Change A2 grade for Reza
10>>> student_grades[1][2] = 100
11>>> print(student_grades)
12[['Khalid', 90, 95, 100], ['Reza', 85, 100, 98], ['Deven', 70, 75, 80]]
