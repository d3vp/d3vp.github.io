New concepts required for this problem set

By default, print() function displays everything on one line

1print("hello")
2print("everyone")
3print()  # print a blank line
4print("how are you?")
Output
hello
everyone

how are you?

We can change this end character using another keyword argument to print() function, end=.

1print("A sequence of numbers:")
2print(1, end=",")
3print(4, end=",")
4print(9, end=",")
Output
A sequence of numbers:
1,4,9,

This is useful in a loop:

1N = 10
2for i in range(N):
3    print(i*i, end=", ")  # comma and a space
Output
0, 1, 4, 9, 16, 25, 36, 49, 64, 81,

in operator (membership operator)

  • In Python in is a keyword.
  • The in and not in operators test for membership.
  • We can use them with strings to test if one string is a substring of another.
1>>> s = "More garbage"
2>>> "age" in s  # True if "age" is found anywhere in s
3True
4>>> "x" in s
5False
6>>> "w" not in s
7True

string.upper() method

1>>> "hello".upper()  # calling method upper() on the string "hello"
2'HELLO'
3

4>>> message = "hello"
5>>> message.upper()  # using variable that refers to string
6'HELLO'
7

8>>> message = 10
9>>> message.upper()  # upper() only available for str objects
10AttributeError: 'int' object has no attribute 'upper'
